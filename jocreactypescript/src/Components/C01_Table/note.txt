
Text Dimensions

0. Row: string
 * SS_TextDimension = 0
 * Row.split => Text 

1. Text: string[]
 * SS_TextDimension = 1
 * Text.merge() => Row 
 * Text.split() => Table 

2. Table: string[][]
 * SS_TextDimension = 2,3
 * Table.merge() => Text 
 * Open Column Tab => setSS_TextDimension(3)
 * Close Column Tab => setSS_TextDimension(2)
 * Name Column as First Row 

Note
1. The output text dimensions of tesseract OCR is controled by SS_TextDimension.
